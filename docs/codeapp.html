Project timeline

4 months (August 2021 - December 2021)

#Code app

#Code app is a learning application for beginners in learning #C. We aim to engage users and have digestible information while learning #C through Study streak, progress trackers, mixed learning materials, downloadable courses, and certification.

It is a course requirement for our Mobile development class. The requirement is for us to develop any Android application and chose to work on what is common for all of the members of the group, learning programming languages.

Users

The application is for Computer science and IT students or anyone who is just starting to learn programming languages.

The problem

Learning programming languages can be daunting and exhausting to learn. For the 3 students I interviewed, this results to procrastination, distress and demotivation.

How might we minimize the negative emotion in studying programming languages?
